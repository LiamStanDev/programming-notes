# 簡介
---
**Overlay Network** 是一種虛擬化的網絡技術，它建立在現有的物理網絡之上，提供邏輯隔離和自訂的網絡層。這類網絡使得**不同的系統或設備能夠在一個虛擬網絡中通信**，而無需直接依賴於底層的物理網絡結構。
### Overlay Network 的工作原理
Overlay Network 由一層虛擬的網絡結構構成，這些虛擬網絡層透過封裝（encapsulation）或隧道（tunneling）技術，在底層的物理網絡之上進行傳輸。底層物理網絡稱為 *underlay network*。Overlay Network 的節點通常使用標準的 IP 協議進行通信，但它們在邏輯上是被隔離的，不受物理網絡的限制。

工作流程：
封裝（Encapsulation）：Overlay 網絡中的數據包會被封裝成**包含額外標頭的封包**，這些標頭包含將數據包路由到正確目標的資訊。這些封裝的數據包會在物理網絡中傳輸。
隧道（Tunneling）：在傳輸過程中，封裝的數據包通過隧道技術在底層物理網絡中進行路由。
