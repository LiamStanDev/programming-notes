# 變數
---
### 變數
```shell
# 定義變量
# 變量
your_name="liam"
# 常量
LD_LIBRARY_PATH="/bin/"
# 數組
my_arr=(1 2 3 4 5) # 需要以空格分割

# 使用變量
echo $your_name # 可以直接使用
echo ${your_name} # 顯示使用 (建議)

# 刪除變量
unset your_name

# 特殊變量
echo $? # 上一個程序退出狀態碼
echo $# # 傳遞給腳本的參數數量

```

### 字符串
* 單引號: 原樣字串
* 雙引號: 可以使用轉義字符與變量
```shell
str1='this is a string'
str2="Hello, ${your_name}\n"

# 字串長度
echo ${#str1}

# 子字串
echo ${string:1:4}
```

### 數組
> 注意 bash 起始 index 為 0，但 zsh 為 1
```shell
array_name=(val0 val1 val2 val3)
echo ${array_name} # 整個數組
echo ${array_name[1]} # 第二個元素
```



# 運算符
---
### 關係運算符

| 運算符   | 說明            | e.g.            |
| ----- | ------------- | --------------- |
| `-eq` | equal         | `[ $a -eq $b ]` |
| `-ne` | not equal     | `[ $a -ne $b ]` |
| `-gt` | greater than  | `[ $a -gt $b ]` |
| `-lt` | less than     | `[ $a -lt $b ]` |
| `-ge` | greater equal | `[ $a -ge $b ]` |
| `-le` | less equal    | `[ $a -le $b ]` |
### 邏輯運算符

| 運算符    | 說明  | e.g. |
| ------ | --- | ---- |
| `&&`   | 與   |      |
| `\|\|` | 或   |      |
