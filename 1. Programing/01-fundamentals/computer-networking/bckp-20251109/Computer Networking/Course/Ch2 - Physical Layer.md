# 通信基礎
---
物理層目的是解決傳輸bit到各種計算機的問題，需要**定義接口特性**與**定義標準**。

## 三種通信方式
信道是信號的傳述媒介本身是單向的。
* Simplex Communication (單工通信): 你說我聽
	* 需要一個信道
	* e.g. 廣播
- Half-Duplex Communication (半雙工通信)：你說一句我說一句
	- 需要兩個信道，但任一方都不能同時發送與接收
	- e.g. 對講機
- Full-Duplex Communication (雙工通信)：我們一起說
	- 需要兩個信道。
	- e.g. 打電話

## 編碼與條制
message(圖片、文字等) -> data(二進制) -> signal (電磁波) -> baseband signal (尚未經過編碼or調制的低頻率電磁波) 
* baseban signal (基帶信號): 又可以分為數字基帶信號(方波)與模擬基帶信號(非方波)
> 在計算機網路中常見的是將**數字基帶信號**通過編碼or條制在不同信道中傳輸。

### 編碼 (Encoding)
表示將基帶信號進行轉換使其可以利用數字信道進行傳輸。
數字基帶信號 : Manchester coding (Enthenet), 4B/5B, 8B/10B
* [-] 模擬基帶信號 : PCM(應用在音頻)

### 調制 (modulation)
表示將基帶信號進行轉換使其可以利用模擬信道進行傳輸。
數字基帶信號 : wifi中使用的CCK, DSSS, OFDM 
- [-] 模擬基帶信號 : FDM

## 數據交換方式
若有n台主機就實現通信需要$C^n_2$ 條線路，這樣是很低效的，讓所有的主機接上同一個交換機，通過一個交換機可以使線路變成n個。有以下三種交換方式：
### 電路交換 (Circuit Exchange)
建立連接 -> 數據傳輸 -> 釋放連接
![[Pasted image 20230808162609.png]]
* 優點：
	* delay小
	* 順序傳送，無失序問題
	* 實時性強
	* full-duplex communication
	* 控制與實現簡單
* 缺點：
	* 建立連接耗時
	* 線路獨佔
	* 某節點損壞需要重頭建立連接
	* 無法緩存，大量數據湧入時無法應付
	* 難以規格化，因為數據直達，無法適應每個節點的接受速率不同。
	* 通信中無法進行差錯控制，只能到目的地時才能發現。
### 報文交換 (Message Exchange)
Message表示一個端點一次要發送的數據單位（包含數據以及目的地址），每個長短不一長度不限。Message Exchange中交換機會接手整個報文，進行差錯控制然後通過路由信息(每個交換機都有路由表)找到下一個轉發的交換機，一直持續到數據傳輸結束。簡單來說就是**存儲轉發**(存儲然後轉發)。
* 優點：
	* 無需建立連接
	* 動態選擇最佳路徑
	* 高可靠性：某交換機壞了，也能即時搜索新的路徑
	* 提高線路力用率
	* 多目的地轉發：一個報文可以發送給多個目的地
	* 規格化：緩存可以處理多節點接受速率不同
* 缺點：
	* 實時性差：存在轉發delay
	* 報文大長度不一：若緩存大小，可能要將報文存放在磁盤導致I/O讀寫的Delay。
### 分組交換 (Packet Exchange)
是現在網路使用的交換方式，會**限制傳輸數據的長度**，一般為128B，也是採用存處轉發的方式，會將報文猜分成一定長度的Packet(組、數據報)，以組為單位進行傳輸。packet包含分割的數據, 目的地址與編號，編號用於接收者組成完整報文。
* 優點：與報文交換相同，還有以下幾個
	* 緩衝區大小固定
	* 加速傳輸：將一個報文猜分成小組並行傳輸。
	* 出錯時重發的數據量小。
* 缺點：
	* 接收端需要更強處理能力：因為需要拼裝組
	* 降低通訊效率：因為要添加編號
	* 存在失序、丟失或者重複分組等問題。
* 特點：
	1. 提供網路層無連接服務
		🔥 這邊的無連接表示不會先指定連接路徑，與TCP, UDP無關
	2. 會有亂序、重複與丟失問題
		* 亂序：可通過編號解決
		* 丟失：當交換節點湧入大量數據時，緩存不夠時導致數據丟失，可透過差錯控制使得發送方重新發送


# 傳輸媒介與設備
---
## 傳輸媒介
### 導向性傳輸媒介
需要沿著固定媒介(固定方向)進行傳輸。e.g. 銅線
1. 雙絞線 (Twisted pair)：用於短距離傳輸
	![[Pasted image 20230808172330.png]]
2. 同軸電纜 (Coaxial cable): 用於中距離傳輸
	![[Pasted image 20230808172504.png]]
	* B 用於減少干擾的屏蔽層
	* D 為內導體
3. 光纖 (Optical fiber): 用於長距離傳輸
	![[Pasted image 20230808172813.png]]
	* 當入射角夠大時，就會產生全反射，使得能量損耗很小。
	* 只要大於一定角度就能實現全反射，所以可以同時發送多個光訊號。
		* 一個光訊號：單模光纖，不會互相干擾，超遠距離傳輸。
		* 多個光訊號：多模光纖，會互相干擾，常距離傳輸。
	* 光纖通常會捆成一個粗線叫做光纜，一條光纜可能有數千條光纖。
### 非導向性傳輸媒介
在自由空間中進行傳輸。e.g. 空氣
1. 無線電波：向四面八方傳播
2. 微波：有方向性
3. 紅外線：有方向性

## 設備
### 中繼器 (Repeater)
由於信號在信道中會有一定程度的衰減，到一定程度會導致數據失真，所以中繼器會將信號放大與還原，增加信號傳輸的距離。
