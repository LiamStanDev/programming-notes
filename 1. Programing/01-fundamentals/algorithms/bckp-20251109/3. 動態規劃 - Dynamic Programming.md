### 甚麼是動態規劃(Dynamic Programming)?
動態規劃**通常是求極值**，也就是最優化問題，核心就是窮舉。有以下三個要素
1. 狀態轉移方程式: 要優先找出來，才能進行窮舉。
2. 是否具備最優子結構: 能否通過子問題最佳達到整個問題最佳。
	最優子問題: 要求子問題之間相互獨立，例如考分數只要每科都是最高分，總分一定是最高分的，若成績互相制約則不是。
3. 重疊子問題: 要解決子問題重複計算的問題。(需要備忘錄(Aarry)或者DP table(Hash table))

#### 概念連結
狀態轉移方程式 -> 解決如何窮舉
重疊子問題 -> 解決如何聰明的窮舉
#### 解題步驟
1. 明確"base case"
2. 明確"狀態"
3. 明確"選擇"
4. 定義函數

#### 兩種模式
##### 自頂向下: 遞迴
> 選擇計算狀態。
```python
def dp(狀態1, 狀態2, ...):
	# base case

	# 選擇
	for 選擇 in 所有可能選擇:
		res = 求極值(res, dp(狀態1, 狀態2, ...))
	return res
```
##### 自底向上: 遞推
> 直接由狀態出發。
```python
# base case
dp[0][0][...] = base case
# 狀態轉移
for 狀態1 in 狀態1下所有取值:
	for 狀態2 in 狀態2下所有取值:
		for ...:
			dp[狀態1][狀態2][...] = 求極值(選擇1, 選擇2, ...)
```
