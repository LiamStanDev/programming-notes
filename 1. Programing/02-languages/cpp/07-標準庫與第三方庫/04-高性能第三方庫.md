# 高性能第三方庫

介紹 HFT 系統常用的高性能第三方庫。

---

## 1. fmt - 格式化庫

### 1.1 基本使用

```cpp
#include <fmt/core.h>
#include <fmt/chrono.h>

// 比 std::cout 快 10x
fmt::print("Price: {:.2f}, Volume: {}\n", 100.12345, 1000);

// 格式化到 string
std::string msg = fmt::format("Order {} filled", order_id);

// 時間格式化
auto now = std::chrono::system_clock::now();
fmt::print("Time: {}\n", now);
```

## 2. spdlog - 高性能日誌

### 2.1 異步日誌

```cpp
#include <spdlog/spdlog.h>
#include <spdlog/async.h>
#include <spdlog/sinks/basic_file_sink.h>

void setup_logging() {
    // 異步日誌 (不阻塞關鍵路徑)
    auto logger = spdlog::basic_logger_mt<spdlog::async_factory>(
        "async_logger", "logs/trading.log");
    
    spdlog::set_default_logger(logger);
    spdlog::set_level(spdlog::level::info);
}

void log_trade() {
    spdlog::info("Order filled: price={}, qty={}", 100.0, 1000);
}
```
