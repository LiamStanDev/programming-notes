C++11 提供操作時間的 chrono 庫，從語言及別提供支持。UNIX 系統從 1970/1/1 作為開始點，從 1970/1/1 經過的秒數用一個整數進行存放。
### 函數與類型
#### time_t 別名
`time_t` 表示時間類型，他是 long 的別名，定義在 `<time.h>` 頭文件中，表示從 1970/1/1 經過的秒數。
##### 聲明
```cpp
typedef long time_t;
```

#### time() 函數
用來獲取操作系統當前的時間，定義在 `<time.h>` 頭文件中。
##### 聲明
```cpp
time_t time(time_t *tloc);
```
##### 使用
```cpp
// 使用方式一
time_t now = time(nullptr); 

// 使用方式二
time_t now;
time(&now);
```

#### tm 結構體
`time_t` 為 long 類型不符合人使用習慣，需要轉換成 `tm` 結構體，定義在 `<time.h>` 頭文件中。
##### 聲明
```cpp
struct tm
{
	int tm_year; // 年分
	int tm_mon; // 月份，取值為 [0(Jan), 11(Dec)]
	int tm_mday; // 日期，取值為 [1, 31]
	int tm_hour; // 小時，取值為 [0, 23]
	int tm_min; // 分鐘，取值為 [0, 59]
	int tm_sec; // 秒，取值為 [0, 59]
	int tm_wday; // 星期，取值為 [0(周日), 6(周六)]
	int tm_yday; // 秒，取值為 [0, 59]
	int tm_isdst // 秒，取值為 [0, 59]
	int tm_isdst; // 夏令標示符號
}
```

#### localtime() 
將 `time_t` 轉換成 `tm` 結構體，定義在 `<time.h>` 頭文件中。
* `localtime()`: 非線程安全。
* `localtime()_r`: 線程安全。
```cpp
int main()
{
	time_t now = time(nullptr);
	std::cout << now << std::endl;

	tm tmnow{};
	localtime_r(&now, &tmnow);
}
```

#### mktime() 
與 `localtime()` 相反，會將 `tm` 轉為 `time_t`，定義在 `<time.h>` 頭文件中。
##### 聲明
```cpp
time_t mktime(struct tm *tm);
```
