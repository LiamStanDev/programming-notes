### .Net APIs
均以 System 作為命名空間，有以下重要的類命名空間:
1. System: 基礎類型、接口、特性、異常、委託與事件等
2. System.Collections: 舊的集合
3. System.Collections.Generic: 新的集合
4. System.Collections.Concurrent: 線程安全集合，包含泛型
5. System.Diagnostics: 提供與系統(進程、日誌)，進行交互還有性能計數器。
6. System.IO: 提供讀寫功能
7. System.Net: 提供網路基礎功能
8. System.Text: 提供文字編解碼功能
9. System.Threading: 提供多線程支持
10. System.LINQ: 提供 LINQ 功能
11. System.Math
12. System.Numeric
13. System.Reflection
