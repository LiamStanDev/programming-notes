授權表示有能允許能訪問某些資源，授權必須在驗證之後。
#### 1. MVC 方式
這種方式比較簡單，只能用於 MVC
* Program.cs
```cs
// ...
app.UseAuthentication();
app.UseAuthorization();
```

* 受保護的端點
```cs
[ApiController]
[Route("api")]
[Authorize] 
public class HomeController : ConstrollerBase
{
	[AllowAnnoymous]
	public IActionResult GetNoProtectedData() {}

	[Authorize(Role="Sales")]
	public IActionResult GetProtectedData() {}
}
```

#### 2. Policy-based
更現代且推薦的方式
* Program.cs
```cs
service.AddAuthorization(opt => 
{
	opt.AddPolicy("AuthPolicy1", p => 
	{
			p.RequireAuthenticatedUsers = true;
			p.RequireClaim("department", "sales");
			p.RequireClaim("status", "senior");
	})
})
// ...
app.UseAuthentication();
app.UseAuthorization();
```

* 受保護的端點
```cs
[ApiController]
[Route("api")]
public class HomeController : ConstrollerBase
{
	public IActionResult GetNoProtectedData() {}

	[Authorize("AuthPolicy1")]
	public IActionResult GetProtectedData() {}
}
```
