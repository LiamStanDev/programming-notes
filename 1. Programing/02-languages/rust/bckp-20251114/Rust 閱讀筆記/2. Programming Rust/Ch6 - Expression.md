### Rust is an Expression Language
Rust is what is called an expression language. This means it follows an older tradition, dating back to Lisp, where expressions do all the work. In Rust, if and match can produce values. Most of the control flow tools in C are statements. In Rust, they are all expressions.
**Rust 為表達式語言**，他遵循了古老的 Lisp 傳統，即表達式負責所有的工作。在 Rust 中 if 與 match 可以產出值。C 語言中所有流程控制都是 statement，而在 Rust 中流程控制均為 expression。
